# Query Optimization Benchmarks

OData query performance analysis including:
- Simple queries vs complex filters
- Filter complexity impact
- Select field optimization
- OrderBy performance
- Pagination overhead
- Count query efficiency

## Files

- `query_bench_test.go` - Query building and execution
- `filter_complexity_bench_test.go` - Filter expression analysis
- `pagination_bench_test.go` - Pagination and skip/top optimization
